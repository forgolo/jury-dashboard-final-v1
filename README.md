# Jury Dashboard Premium

A high-performance dashboard for jury evaluation, built with glassmorphism and real-time Firebase synchronization.

## Features
- Real-time scoring and synchronization
- Expert evaluation interface
- Coordinator dashboard with CSV export
- Dark/Light mode support
- Mobile responsive design

## Technologies
- HTML/CSS/JS (Vanilla)
- Firebase Realtime Database
- SheetJS (XLSX) for data import
