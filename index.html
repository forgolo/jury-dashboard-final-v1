Подключение Firebase

Для синхронизации оценок между устройствами нужна база данных Firebase. Это бесплатно.

1
Создайте Firebase-проект
Перейдите на console.firebase.google.com → «Добавить проект» → название → отключите Analytics → «Создать»
2
Создайте Realtime Database
Левое меню: «Сборка» → «Realtime Database» → «Создать базу данных» → выберите регион → «Начать в тестовом режиме» → «Включить»
3
Скопируйте конфигурацию
Шестерёнка → «Настройки проекта» → «Ваши приложения» → «Добавить приложение» → Веб () → придумайте псевдоним → «Зарегистрировать» → вкладка «JSON» → скопируйте объект firebaseConfig
4
Вставьте конфигурацию ниже
Нажмите «Подключить» — конфигурация сохранится в браузере
